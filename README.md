# Tradução PT-BR para ReIndev
Ela está pronta, pode baixar e jogar o jogo traduzido!

Caso haja algum erro na tradução, por favor me avise!

Ainda tenho planos de traduzir os "Splashes", mas por enquanto está pronta.

## Instalação
1. Baixe o .zip em https://github.com/pandorasarah/ReIndev_PT-BR/releases/latest
2. Coloque os arquivos na pasta "languagepacks" dentro da instância do ReIndev.
3. Selecione o idioma dentro do menu do jogo.
