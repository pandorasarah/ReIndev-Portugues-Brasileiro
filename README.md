# Tradução PT-BR para ReIndev

## Sobre a Tradução
Ela não está finalizada, muita coisa ainda precisa ser traduzida!
Diria que o Menu e uns 60% dos Blocos já foram traduzidos. 
Falta Itens, Conquistas, Dicas, Créditos e as "Splashes".

## Instalação
(Farei um guia melhor, ainda sou meio nova no github :P)
1. Baixe os arquivos "pt_BR.lang", "credits-pt_BR.txt", "splashes-pt_BR.txt" e "tips-pt_BR.txt".
2. Coloque os arquivos baixado na pasta "languagepacks" dentro da instância do ReIndev.
3. Selecione o idioma dentro do menu do jogo.
